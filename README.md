# GYMFLOW 💪

週の頻度に合わせた筋トレメニュー管理PWAアプリ

## 機能
- 週1〜6回の頻度に合わせたプログラム自動生成
- 筋肉回復を考慮した曜日スケジュール提案・編集
- 前回の最高重量（PR）記録・表示
- 種目ごとのYouTube解説リンク・代替種目

## セットアップ

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## iPhone PWAとして使う方法

1. Vercelでデプロイ後、iPhoneのSafariでURLを開く
2. 画面下の共有ボタン（□↑）をタップ
3.「ホーム画面に追加」を選択
4. アプリとして起動できるようになります
